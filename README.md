# 🎉 CheerUp Corner

> A cozy little corner on the internet to express your feelings, save your memories, and brighten your day.

## 🌟 Overview

CheerUp Corner is a mood-based web application that allows users to explore content tailored to their emotions. Whether you're feeling happy, sad, calm, excited, angry, or romantic, the app provides mood-specific thoughts, jokes, and uplifting messages to make your day a little better.

Users can also create and save their own personal stories for each mood, making CheerUp Corner a private space to reflect on emotions and memories.

## ✨ Features

- 😊 Select from different moods
- 📖 Get mood-based thoughts, jokes, and uplifting content
- 📝 Write and save personal mood stories
- 🔒 Private user accounts with authentication
- 📧 Email verification during signup
- 📚 View, edit, and delete saved stories
- 📱 Fully responsive design for desktop and mobile
- 🌙 Modern and clean UI

## 🚀 How to Run Locally

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/CheerUpCorner.git
cd CheerUpCorner
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Create Environment Variables

Create a `.env.local` file in the root directory and add:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 4. Start the Development Server

```bash
npm run dev
```

### 5. Open in Browser

Visit:

```txt
http://localhost:3000
```
